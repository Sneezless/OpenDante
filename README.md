# OpenDante

OpenDante is an open-source, clean-room reimplementation of the RPG Maker Dante 98 runtime.

The goal is to preserve the original engine by providing a modern, cross-platform runtime capable of loading and executing original Dante 98 game data without modification.

This project is primarily focused on documentation, reverse engineering, compatibility, and preservation.

## Goals

- Load original RPG Maker Dante 98 projects
- Execute original event scripts
- Render original graphics and maps
- Preserve original engine behaviour where practical
- Document Dante 98 file formats and runtime behaviour
- Provide tools for inspection, extraction and debugging

## Non-goals

OpenDante does **not** distribute or contain copyrighted RPG Maker Dante 98 assets, BIOS ROMs, or commercial game data.

Users must provide their own legally obtained game files.

## Planned Applications

- OpenDante Player
- Map Viewer
- Event Viewer
- Resource Extractor
- Save Editor
- Event Debugger

## Project Structure

```
apps/

```

## Current Status

Early research.

Nothing is currently implemented.

Initial work focuses on:

- collecting documentation
- understanding file formats
- creating test projects
- documenting event opcodes
